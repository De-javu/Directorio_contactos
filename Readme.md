# Gestor de Contactos - Práctica Python

## Pasos para configurar y ejecutar el proyecto

1. **Crear entorno virtual**
   - Se crea la carpeta `venv` con el entorno virtual de Python:
``
     python -m venv venv
     ```

2. **Activar el entorno virtual**
 - En Linux/Mac:
    ```
      source venv/bin/activate
     ```
   - En Windows:
     ```
     bash
     source venv/Scripts/activate
     ```
     o
     ```
     .\venv\Scripts\activate
     ```

3. **Instalar dependencias**
   - Se instalan las librerías necesarias (si las hubiera) usando:
     ```
     pip install -r requirements.txt
     ```
   - Para este proyecto, solo se usan librerías estándar (`csv`, `json`), así que este paso es opcional.

4. **Estructura del proyecto**
   - El proyecto contiene:
     - `main.py`: archivo principal con el menú y la lógica principal.
     - `contactos.py`: funciones para gestionar los contactos.
     - `contactos.csv` y `contactos.json`: archivos generados para guardar los contactos.
     - `proyecto_gestor_contactos.md`: enunciado y requisitos de la práctica.

5. **Ejecución**
   - Para ejecutar el programa:
     ```
     python main.py
     ```
